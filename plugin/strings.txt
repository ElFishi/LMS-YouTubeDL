	DE	YouTubeDL
	EN	YouTubeDL

PLUGIN_YOUTUBEDL_DESC
	DE	Lädt YouTube-Audio per yt-dlp herunter. Stellt CLI/JSON-RPC-Befehle bereit: youtube download url:<url> und youtube download log.
	EN	Downloads YouTube audio via yt-dlp. Provides CLI/JSON-RPC commands: youtube download url:<url> and youtube download log.

	EN	yt-dlp binary

PLUGIN_YOUTUBEDL_SETTINGS_YTDLP_DESC
	DE	Pfad zum yt-dlp Binary. Leer lassen, um das Binary des YouTube-Plugins zu verwenden (empfohlen). Gib einen absoluten Pfad ein, um ein eigenes Binary zu verwenden.
	EN	Path to the yt-dlp binary. Leave blank to use the binary from the YouTube plugin (recommended). Enter an absolute path to use a custom binary.

PLUGIN_YOUTUBEDL_SETTINGS_FFMPEG
	DE	ffmpeg-Pfad
	EN	ffmpeg path

PLUGIN_YOUTUBEDL_SETTINGS_FFMPEG_DESC
	DE	Pfad zur ffmpeg-Binary (benötigt für Audioextraktion, Thumbnail-Einbettung und Konvertierung). Leer lassen, um ffmpeg aus dem System-PATH zu verwenden. Gib einen absoluten Pfad ein, wenn ffmpeg nicht im PATH ist.
	EN	Path to the ffmpeg binary (required for audio extraction, thumbnail embedding and conversion). Leave blank to use ffmpeg from the system PATH. Enter an absolute path if ffmpeg is not on PATH.

PLUGIN_YOUTUBEDL_SETTINGS_DOWNLOAD
	DE	Download-Einstellungen
	EN	Download settings

PLUGIN_YOUTUBEDL_SETTINGS_DOWNLOAD_DESC
	DE	Legt fest, wohin yt-dlp heruntergeladene Audiodateien speichert. Felder leer lassen, um die Standardwerte zu verwenden. yt-dlp-Ausgabe-Templates (z. B. %(title)s) werden unterstützt.
	EN	Configure where yt-dlp saves downloaded audio files. Leave fields empty to use the defaults. yt-dlp output templates (e.g. %(title)s) are supported.
